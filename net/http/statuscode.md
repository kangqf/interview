### HTTP 状态码

首先 2xx 系列标识正常， 3xx 系列标识重定向，4xx标识客户端请求错误，5xx标识服务器响应错误

1. 200
  正常响应
2. 304 
  内容未修改，直接读取本地的缓存
  ![](/assest/img/http304.png)
3. 401
  未鉴权
4. 403 
  未授权
5. 404 
  内容未找到    
6. 204
  请求执行成功，但是无数据返回，例如将一个文件PUT到服务器成功后就返回204以表征文件放置成功。使用   DELETE  删除一个文件也是一样的道理
